The data used in this section is related to the climate approach. It can be retrieved from https://wutwaw-my.sharepoint.com/personal/piotr_palka_pw_edu_pl/_layouts/15/onedrive.aspx?id=%2Fpersonal%2Fpiotr%5Fpalka%5Fpw%5Fedu%5Fpl%2FDocuments%2FProjekty%2FTerratransformacja%20Marsa%2FGCM%20data&ga=1. In particular, only the folder every_month is used.
Workflow:

1. Visit the link
2. Acces the site through your university mail. If not possible, visit the nasa climate database for a richer but perhaps not perfectly correlated data sets to those used in this project.
3. Retrieve every file from the folder every_month.
4. Create an appropriate directory hierachy using as refernce the files in /model.
